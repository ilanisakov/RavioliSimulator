# Volos Twitter Bot

# Installation
```
npm install             // Install necessary NPM packages
npm install pm2 -g      // Install pm2 globally
```
# Run

To run in the forground
```
node index.js
```

To run in the background
```
pm2 start index.js
```

To stop the bot from running in the background
```
pm2 stop 0
```

Enjoy
